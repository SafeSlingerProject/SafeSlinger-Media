SafeSlinger Media Sources
=================

Media sources used for the SafeSlinger messenger and key exchange project. Here you can find shared resources like icons and web pages. This is also a great place for general questions about the project. 

- **/icons** Source graphic and icons for all projects.
- **/web** Project website source code and graphics.
- [**/wiki**](https://github.com/SafeSlingerProject/SafeSlinger-Media/wiki) Loads of information and support for all SafeSlinger projects.
