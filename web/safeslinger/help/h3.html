How to Change Default Account in Contact Access.

This page helps iOS users to change their default account access for SafeSlinger.


Step1: Select Settings to configure the default access account for contact.

Step2: Select 'Mail, Contacts, Calendars'.

Step3: Scroll down to the Contacts Section and Check the Default Account field.

Step4: Change it to 'On My iphone' or 'iCloud' if you already enabled icloud feature.
