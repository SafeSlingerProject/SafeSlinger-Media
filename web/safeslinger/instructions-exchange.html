After downloading the appropriate app for your Android device or iOS device, follow these instructions to use SafeSlinger.

1. Ensure all members in proximity have SafeSlinger installed.

2. Start the application.

3. Select yourself from your address book. Once you do so you will see your information fields appear on the next screen.
4. Select the fields you want to share, then tap the "Begin Exchange" button. Android mobile device screen shots Apple mobile device screen shots

 
5. Select the number of users in the group.      

 
6. Enter the lowest common User ID for all members of your group to use for this exchange, then press "OK".

 
7. Of the three word lists one will be common in your group. Select the common word list and press "OK". If you cannot find a common list, tap "No Lists Match".

 
8. Select the contacts you wish to save and press "Done".
