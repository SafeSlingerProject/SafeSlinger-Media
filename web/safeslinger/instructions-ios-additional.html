After downloading the appropriate app for your Android 2.2+ device or iOS 6.0+ device, follow these instructions to use SafeSlinger.

1. Start the application.

2. Select yourself from your address book (for first-time users).
3. Enter a passphrase you would like to use to protect your private key with.  First-time users will need to do this twice for confirmation. Android mobile device screen shots

 
4. Selecting the Sling Keys tab will allow you to exchange public keys and other contact data.  See the exchange instructions page for details.

 
5. Selecting the Compose tab will allow you to select any recipient you have performed the above exchange with, and send a file and/or text message.

 
6. Selecting the Messages tab will allow you to review any messages sent or received.  Tapping on any message will allow you to show or hide the message details.  Long press on the message to download any attachments or remove the message.
